Current Time

.

.

 The code on this webpage displays the current day of the week and the time that the browser loaded the webpage. It is important to note that the client performed the JavaScript on this webpage. In other words, the time that the webpage is displayed is dependent on the client's time. The browser displays the time the JavaScript found from the client's time settings. In other words, the time displayed relies upon the time settings of the client machine. This script has several real-world applications:

 Firstly, the day of the week lets the user know what day it is. This is useful to the user. However, more importantly, this information can be extended to secondary scripts. For example, if it is Wednesday, a website could highlight the office hours for Wednesday.

 Secondly,The current date is also useful. A simple JavaScript could tell the user:

a) How many days are left until an assignment is due.
b) It could tell the user many days until an event occurs.
c) If today is the user's birthday. Or,
d) There are many other applications.
