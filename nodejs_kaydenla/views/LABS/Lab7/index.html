HTML Form POST

. . . Lab 7 Description

 This lab demonstrates basic usage of HTML forms.
This lab uses the POST inside of the HTML form field in order to push the given user responses to the an external .php website. This, of course has many use cases and objectives. For starters, this is useful as a request page. This page demonstrates a page where users fill out a form where appointment requests are made.
One will notice that this page also demonstrates the usage of several other commonly found form functions. The usage of declaring the type of input inside the input fields enables the usage of system-specific feilds. The time type appears differently based on operating system / web browser features. This can enrich the webpage without adding excess load-time / bandwidth.
