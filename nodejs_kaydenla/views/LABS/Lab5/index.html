Get File Extension
 Welcome to Kayden’s Lab 5. This lab demonstrates several basic concepts. The basic concepts delivered here can be applied in many differing use-cases. The code on this webpage asks the user to input the full name of a file (including the file extension) and thus returns the file extension of the user's input.
To achieve this, the user's input is indexed with the last position of a period, and anything coming after the period is accepted as a valid file extension.
To minimize errors, get_extension.js has several error catching checks built in.

Basic Functionality

1. The user is proveded with an input box.
2. User input is thus passed to a variable.
3. The variable is thus capable of being iterated upon with JavaScript.

Use-Cases

1. User input could be used for naming files, events, and any other namable object.
2. Calculation and iteration is possible using the combination of the input box and JavaScript.
3. User input could be used to perform search and indexing actions.
