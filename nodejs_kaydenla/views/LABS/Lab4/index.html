Current Aboslute Path

.

 Welcome to Kayden’s Lab 4. This lab focuses on JavaScript (.js) file implementation and calling of the current time. The HTML code of this webpage defines an instance of the H3 div as being a “display_absolute_path” div type. This div type element is defined and sequentially calls the display_absolute_path function.

This type of code is useful and can be implemented in several different useful ways.

a) A navigation panel could use the absolute path script to be self-aware of itself.
b) Copy right footer notes could display the URL of the current webpage. The navigation panel script could thus adapt and display only relevant information. The navigation text for the current webpage could be either hidden or highlighted to depict that this is where the user currently resides.
