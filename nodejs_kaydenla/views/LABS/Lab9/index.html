. . . Lab 9 Description

 Welcome to Kayden’s Lab 9.
Within this lab one will find a live display of a clock. This clock is created by using the JavaScript datetime module. The live-update nature of the clock on this webpage demonstrates the ability of JavaScript live-updating a webpage. Keeping JavaScript running and updating every second on a webpage has many potentials. This could be used as a countdown timer for a student taking a test. This could be used as a countdown timer for a live auction listin.
