. . . Lab 8 Description

 Welcome to Kayden’s Lab 8.
This lab meets deliverables in the form of Create/Change/ Password page. The password verification is done through the use of JavaScript string-matching. To make the code easier to condensed, the script was written in JQuery.
It is important to note that password verification is the deliverable here. Password complexity requirements are added to enhance the deliverable.
It is heavily noted that this is only the bare-bones form of a functioning page to set a password. It is highly recommended that extra encryption be done on top of HTTPS encryption. This is to prevent sniffers from accessing clear-text, assuming that they break through the session ID.
P.S.The JQuery code and the HTML form code are slightly redundant. This is to maximize compatiability for devices running without JavaScript support.
