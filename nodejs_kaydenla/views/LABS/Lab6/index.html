JQuery String Formatting
 Welcome to Kayden's Lab 6. This lab demonstrates basic usage of JQuery.
In essence, JQuery is a shorthand version of JavaScript. The usage of JQuery results in the same use-cases and performance as using standalone JavaScript. However, the big difference comes into play when considering code complexity.
Performing simple actions with JavaScript can be time consuming and can be a lenghy process. The usage of JQuery simplifies this process for developers- both those that create the code and also toward those that later edit the code.
